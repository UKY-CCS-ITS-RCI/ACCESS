Software Name: VisiData
Overview: VisiData is an interactive multitool for tabular data. It is a terminal-based application for exploring and arranging tabular data in unix-like pipelines. 
Core Features: 
- Navigate the file system and enter directories to load the directory itself as a directory sheet.
- Edit values and text in place.
- Set widths of all visible Columns to their content with Shift+W.
- Move the current column on the left.
- Add, delete, modify and move columns.

General Tags: Data Analysis, Data Visualization, Tabular-Data, Terminal-based
