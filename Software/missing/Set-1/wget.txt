Use Case: Use for downloading files from the internet via the command line.
Code Details and Examples: Code 

Example:

wget https://example.com/file.iso

This command tells wget to download the file from the given URL. If the file is a large one, the download can be resumed at a later time, even if the internet connection was lost during the download.
